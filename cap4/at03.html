Seção 2 - Ranking de Tecnologias

  1. HTML

    Responsável pela estrutura das páginas web.

  2. CSS

    Controla o estilo e o layout dos sites.

  3. JavaScript

    Adiciona interatividade e comportamento às páginas.

  4. API

    Permite a comunicação entre diferentes sistemas.

  5. Git

    Ferramenta essencial para controle de versões em projetos.

Seção 3 - Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.

CSS

JavaScript
Linguagem de programação usada para adicionar interatividade.

DOM
Representação dos elementos HTML em forma de árvore.
API
Interface de Programação de Aplicações
Conjunto de regras que permite a comunicação entre sistemas.
HTTP
Protocolo utilizado para transferência de dados na Web.
URL
Endereço Web
Localização de um recurso disponível na Internet.
DNS
Sistema que traduz nomes de domínio em endereços IP.
